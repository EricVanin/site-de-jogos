# Project Overview (MVP)

## Objetivo
Construir um site web mobile-first de jogo da velha com multiplayer online por codigo de sala, cinco modos de jogo e monetizacao por anuncios.

## Metas de produto

- Partidas 1v1 em tempo real estaveis e rapidas.
- Regras claras para cada modo.
- Baixa friccao de entrada (guest-only).
- Revanche em 1 clique.

## Metas de negocio

- Validar retencao inicial com modos variados.
- Gerar receita com `banner` + `intersticial` a cada 3 partidas concluidas.

## Escopo do lancamento

- 5 modos no MVP:
  - Classico 3x3
  - Sem Velha (pecas somem)
  - Poderes (deck)
  - Tabuleiro 5x5 (vitoria em 4)
  - Rounds BO5 (rotacao de modos)

## Fora de escopo no MVP

- Ranking global.
- Login obrigatorio.
- Matchmaking aleatorio.
- Chat em sala.
